# HomeMind — 你的全知全能 AI 家庭生活管家 🏠🤖

> “不是你控制家，而是家懂得你。”

## 🌟 项目愿景 (Vision)

传统智能家居依然停留在“指令-执行”的被动时代（如“帮我开灯”、“把空调调到26度”）。
**HomeMind** 致力于打破这种僵局，打造一个真正主动、懂你的全知全能型 AI 生活管家。

它通过模拟接入摄像头视觉感知、智能设备传感器、手机定位（GPS）、以及未来规划的车机联动，持续在后台学习每位家庭成员的生活习惯。**你不需要开口，它已经知道你需要什么，并主动为你提供服务。**

## ✨ 核心能力 (Core Capabilities)

### 1. 🧠 记忆与深度学习 (Memory & Learning)
*   **物品轨迹追踪**：忘记钥匙放哪了？不知道文件塞在哪？HomeMind 会默默记录你随手放置物品的最后位置，一问便知。
*   **家庭成员作息建模**：自动区分爸爸、妈妈和小孩的不同作息与偏好。知道你几点下班，知道你周末喜欢睡懒觉。
*   **食材生命周期管理**：自动记录冰箱存入的食材与购买日期，发现快过期的食材（如买了3天的茄子和豆腐）会主动提醒并推荐菜谱。

### 2. ⚡ 主动智能服务 (Proactive Intelligence)
*   **无感环境预调**：下班路上还有10分钟到家？它会根据当前室外温度，提前为你开启暖气或空调，并在车机上播放你喜欢的音乐。
*   **场景无缝切换**：检测到周末一家人坐在沙发上，自动触发“影院模式”（拉上窗帘，灯光调至暖橙色15%亮度）。
*   **异常安全感知**：深夜听到奇怪声音？HomeMind 若检测到只是扫地机器人滑落，会主动开走廊灯并温柔播报，打消你的疑虑。

### 3. 🌐 全场景生态联动 (Omni-scene Ecosystem)
*   **精准硬件控制**：完美联动灯光、空调、暖气、窗帘、扫地机器人、空气净化器、门锁、热水器、洗衣机等全屋家电。
*   **UI 实时平面图反馈**：提供一个沉浸式的家庭 2D 俯视图，所有 AI 操作（如开儿童房灯、人员移动到书房）均会在平面图上实时渲染动效。
*   **人车家互联（规划中）**：打通手机、汽车与智能家居的边界，实现通勤路上的无缝服务衔接。

## 🛡️ 隐私与安全 (Privacy & Security)

在赋予家“大脑”的同时，我们绝不妥协隐私。
HomeMind 倡导**本地化边缘计算**理念。所有家庭记忆与作息数据均存储在本地，摄像头仅提取关键状态标签（如“有人/无人”、“移动/静止”），原始视频画面绝不上传云端，赋予用户 100% 的数据掌控权。

## 🚀 快速开始 (Getting Started)

### 技术栈
*   **前端**：React + Vite + Tailwind CSS + Framer Motion
*   **AI 引擎**：DeepSeek LLM (支持上下文长记忆与强制标签输出)
*   **可视化**：内联交互式 SVG 平面图

### 安装与运行

1. 克隆仓库：
   ```bash
   git clone https://github.com/chyxin071-sys/homemind.git
   cd homemind
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 配置 AI 密钥：
   在项目根目录创建 `.env.local` 文件，并填入你的 DeepSeek API Key（**注意：此文件已被 `.gitignore` 忽略，绝对不要将 Key 提交到代码库中！**）：
   ```env
   DEEPSEEK_API_KEY=your_api_key_here
   DEEPSEEK_MODEL=deepseek-chat
   ```

4. 启动开发服务器：
   ```bash
   npm run dev
   ```
   （这会同时启动前端界面和后端的 API 代理服务）
